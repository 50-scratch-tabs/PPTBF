PPTBF

Propose Projects To Be Featured

PPTBF (Propose Projects To Be Featured) is a Scratch studio run by the Scratch Team and curated by volunteers, for Scratchers to suggest a project to be featured.

The studio can be found here, and anyone can propose projects in comments section.

There are only two rules for proposing. (Well, really three, but the last one shouldn't need to be said)

  1. The user proposing the project must be a Scratcher.
  2. Users cannot propose projects created by them.
  3. Projects must follow the Scratch Community Guidelines. (duh)
